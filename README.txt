1. В папке files хранится log.txt - результат моиторинга программы.
2. В папке obj  хранится main.c, для запуска скрипта переходим в эту папку и прописываем "gcc main.c lee.c monitoring.c -lpthread -Wall -o program", а затем "./program"
3. В папке src хранятся файлы программы, в которых:
  lee.c - Функция проверки возможности перехода на позицию (строка, столбец), от текущей позиции. Функция возвращает false, если (строка, столбец) недопустимая позиция или имеет значение 0 или уже посещено
  monitoring.c - Функция мониторинга системы
  lee.h & monitoring.h - Программы для запуска соответвствующих функций 
4. Запуск програмы происходит из папки obj, для запуска прописываем "gcc main.c lee.c monitoring.c -lpthread -Wall -o program", а затем "./program", далее вводим координаты начальной и конечной вершин, кратчайший путь между которыми, посчитает программа. На экране выводится кратчайший путь(если он сущетсвует). Результат мониторинга системы будет помещен в файл log.txt в папку files.
